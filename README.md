# Budget-Tracker-cpp
A multi-user budget tracker built in C++ with expense categories, balance calculation, timestamps and persistent file storage.


# MyBudget App

A multi-user budget tracker that helps Nigerians manage 
monthly income and expenses by category.

## Features
- Add multiple expense categories dynamically
- Calculates remaining balance with overflow warning
- Saves sessions with timestamps to personal files
- Multi-user support with welcome back detection

## How to run
1. Compile: g++ budget-tracker.cpp -o budget
2. Run: ./budget

## What I learned
Vectors, nested loops, parallel data structures, 
dynamic filenames and fstream file handling.
